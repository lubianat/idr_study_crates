IDR study metadata RO-Crate profile

This profile describes RO-Crates generated from IDR study metadata files (idrXXXX-study.txt). It documents the conventions used by the converter in this repository.

Core expectations

  • The root dataset uses the IDR study URL as the @id.
  • The root dataset declares conformsTo with this profile URI.
  • Study-level fields are preserved as additionalProperty entries.
  • Screens and experiments are Dataset entities with additionalProperty.
  • Ontology terms are DefinedTerm entities with URI @id values.
